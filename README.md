# Foodly

Foodly е апликација за нарачка на храна и пијалоци од разни места.

## Карактеристики
 * Логин и регистрација преку FirebaseAuth
 * Чување на дата во Firebase Cloud Firestore
 * Локациски сервиси
 * Мапа
 * Камера
 * Navigation bar
 * State Management
 * UI
 * Cart
 * Favorite
 * Featured
